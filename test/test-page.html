Phone number detection test page

Load this page with the GlobalClick2Dial extension enabled. Numbers in green-bordered blocks should become clickable links; red-bordered blocks should not.

Should be detected (valid formats)

US with parentheses: Call us at (555) 123-4567 for support.

US with dashes: 555-123-4567

US with dots: 555.123.4567

US with spaces: 555 123 4567

US plain 10 digits: 5551234567

International with +: +1 555 123 4567 or +44 20 7123 4567

UK style: 020 7123 4567 (if default country is 44)

At start of sentence: 555-987-6543 is the main line.

At end of sentence: The number is 555-987-6543.

After comma: Contact sales, 555-111-2222, for pricing.

In parentheses: Main office (555) 333-4444 is open 9–5.

International 11+ digits with +: +1 555 123 4567

Should NOT be detected (bad examples)

ID with hyphen (no space): MAT-25082189552

Letters attached: abcd5555555efgh

Underscore prefix: REF_5551234567 or Order_ID_5551234567

11+ digits without + (IDs, not phones): 25082189552 or 1-555-123-4567 or 111111111111

Extension suffix (letters after): 555-123-4567ext or 555-123-4567 x100

Part of word: phone5551234567number

Invoice/code style: INV-555-12345678

Long digit string (ID/serial): 123456789012345

Short digit string (too few): 555-123 (only 6 digits)

Edge cases

Hash before (often still valid): Order #555-123-4567

Colon before: Tel: 555-123-4567

Two valid numbers in one line: Call 555-111-2222 or 555-333-4444 for help.

If a number in a green block does not become a link, or a number in a red block does become a link, the detection rules may need adjustment.
