GlobalClick2Call

Configure how phone numbers are turned into clickable links.

Link protocol

Choose the protocol used when converting phone numbers to links.

Domain filter

Control which sites the extension runs on. Use a comma-separated list; wildcards supported (e.g. *.example.com).

Domains where the extension is disabled (blacklist).

Default country code

When a number has no country code, assume this country. Numbers are normalized to E.164 (e.g. +1 for US).

e.g. 1 (US/Canada), 44 (UK), 49 (Germany)
